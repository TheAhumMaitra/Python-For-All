# Python For All
---
A book for every Python developer and all ages. This book helps you to master Python, it's also can be used as a reference, guide or even docs.

## Who should read this book

---

This book is designed for all level developers learn ore revise or view the Python syntax and special features. It is not meant to replace the original documentation. This book is originally designed for all levels Python developers.

## Why I (Ahum) decided to make the book

---

> "I have seen many beginner or intermediate or even myself sometimes to find a suitable platform for
> find new features to Python or existed features easy documentation. I believe "Simplicity is better than complex". Maybe this is book not for everyone but I tried make for everyone. I don't want replace anything. I want make something.
> That's why I decided to create beautiful book"
>
> **~ Ahum Maitra (Author of this book)**
